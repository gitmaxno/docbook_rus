***** Attributes *****
Common_attributes and common_linking_attributes.
***** Description *****
A member is an element of a simplelist. Unlike the other lists, items in a simplelist are constrained to character data and inline elements.
**** Processing expectations ****
Formatted inline. How the inline members are formatted with respect to each other is controlled by the containing simplelist.
**** Parents ****
These elements contain member: simplelist.
