
***** indexterm (db.indexterm.endofrange) *****
indexterm — Identifies the end of a range associated with an indexed term.
***** Synopsis *****
    * empty
***** Attributes *****
Common_attributes and common_linking_attributes.
Additional attributes:
    * class (enumeration)
          o “endofrange”
    * startref (IDREF)
Required attributes are shown in bold.
***** Description *****
An “end of range” indexterm marks the end of a range. It must have an associated “start of range” indexterm. The resultant index entry applies to the entire range. See indexterm_(db.indexterm.singular).
**** Processing expectations ****
See indexterm_(db.indexterm.singular).
**** Attributes ****
Common_attributes and common_linking_attributes.
  class
      Identifies the class of index term
      Enumerated values:
      “endofrangeThe end of a range
  startref
      Points to the start of the range
