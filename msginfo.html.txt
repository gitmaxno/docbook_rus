
***** msginfo *****
msginfo — Information about a message in a message set.
***** Synopsis *****
    * Zero or more of:
          o msgaud
          o msglevel
          o msgorig
***** Attributes *****
Common_attributes and common_linking_attributes.
***** Description *****
The msginfo element provides additional information about a msg in a msgentry.
**** Processing expectations ****
May be formatted inline or as a displayed block, depending on context. Sometimes suppressed.
On the whole, the semantics of msgset are not clearly defined.
**** Parents ****
These elements contain msginfo: msgentry.
**** Children ****
The following elements occur in msginfo: msgaud, msglevel, msgorig.
