
***** package *****
package — A software or application package.
***** Synopsis *****
    * Zero or more of:
          o text
          o phrase (db._phrase)

***** Attributes *****
Common_attributes and common_linking_attributes.
***** Description *****
The package element identifies a package of some sort. The precise kind of package is context dependent. It might be used, for example, to identify a Java package or an operating system distribution package.
