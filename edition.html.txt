
***** edition *****
edition — The name or number of an edition of a document.
***** Synopsis *****
    * Zero or more of:
          o text
          o phrase (db._phrase)
          o replaceable
          o Graphic inlines [[+]] [[-]]
                * inlinemediaobject
          o Indexing inlines [[+]] [[-]]
                * indexterm (db.indexterm.endofrange)
                * indexterm (db.indexterm.singular)
                * indexterm (db.indexterm.startofrange)
          o Linking inlines [[+]] [[-]]
                * anchor
                * biblioref
                * link
                * olink
                * xref
          o Ubiquitous inlines [[+]] [[-]]
                * alt
                * anchor
                * annotation
                * biblioref
                * indexterm (db.indexterm.endofrange)
                * indexterm (db.indexterm.singular)
                * indexterm (db.indexterm.startofrange)
                * inlinemediaobject
                * link
                * olink
                * remark
                * subscript
                * superscript
                * xref
***** Attributes *****
Common_attributes and common_linking_attributes.
***** Description *****
The edition contains the name or number of the edition of the document.
**** Processing expectations ****
May be formatted inline or as a displayed block, depending on context. Sometimes suppressed.
**** Parents ****
These elements contain edition: biblioentry, bibliomixed, bibliomset, biblioset, info (db.info), info (db.titleforbidden.info), info (db.titleonly.info), info (db.titleonlyreq.info), info (db.titlereq.info).
*** Children [[+]][[+]] ***
This element contains 16_elements.
The following elements occur in edition: text, alt, anchor, annotation, biblioref, indexterm (db.indexterm.endofrange), indexterm (db.indexterm.singular), indexterm (db.indexterm.startofrange), inlinemediaobject, link, olink, phrase (db._phrase), remark, replaceable, subscript, superscript, xref.
